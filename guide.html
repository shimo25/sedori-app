1. アプリについて

このアプリは PWA（Progressive Web App） として実装されています。iPhone の Safari でアクセスして「ホーム画面に追加」すれば、ネイティブアプリと同じ使用感で利用可能です。

  • データは iPhone 内の IndexedDB に保存され、外部に送信されません
  • オフラインでも動作（Service Worker）
  • App Store の審査・Apple Developer 年会費は不要

2. 方法1: GitHub Pages で公開（推奨・無料・恒久）

最も安定している方法です。一度設定すれば、どこからでも iPhone で開けます。

手順

  1. GitHub アカウントを作成（未所持の場合）
    https://github.com/signup
  2. 新規リポジトリを作成（例: sedori-app、Public を選択）
  3. Mac のターミナルで以下を実行： 
    cd "/Users/simodaira/Desktop/せどり管理アプリ（β版）"
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/【あなたのID】/sedori-app.git
git push -u origin main
  4. GitHub のリポジトリ画面で Settings → Pages。
    Build and deployment の Source で Deploy from a branch を選び、Branch を main、フォルダは / (root) にして Save。
  5. 数分後、https://【あなたのID】.github.io/sedori-app/ が公開されます。
  6. iPhone の Safari でそのURLを開き、共有 → 「ホーム画面に追加」。
GitHub Pages は HTTPS で配信されるため、Service Worker が正常に動作してオフラインキャッシュが効きます。

3. 方法2: Netlify Drop（ドラッグ&ドロップ・無料）

最も早く公開できる方法。アカウント登録不要で10秒で完了します。

  1. Mac のブラウザで https://app.netlify.com/drop を開く
  2. プロジェクトフォルダをそのままドラッグ&ドロップ
  3. 発行される URL（例: random-name.netlify.app）を iPhone の Safari で開く
  4. 共有 → 「ホーム画面に追加」
無料プランの URL は削除されることがあるので、継続的に使うなら Method 1（GitHub Pages）推奨。

4. 方法3: ローカル Wi-Fi 経由（外部公開したくない場合）

  1. Mac のターミナルで： 
    cd "/Users/simodaira/Desktop/せどり管理アプリ（β版）"
python3 -m http.server 8080
  2. Mac の IP アドレスを確認（システム設定 → ネットワーク → Wi-Fi → 詳細 → TCP/IP）
  3. iPhone を同じ Wi-Fi に接続し、Safari で http://【MacのIP】:8080/ を開く
  4. 共有 → 「ホーム画面に追加」
Mac がスリープ / Wi-Fi から切断されると開けなくなります。あくまで動作確認用です。HTTPS ではないため Service Worker は localhost 扱い以外では無効化されます。

5. ホーム画面への追加手順（iPhone Safari）

  1. Safari でアプリ URL を開く
  2. 画面下の 共有ボタン（四角から矢印が出ているアイコン）をタップ
  3. メニューを下にスクロールして 「ホーム画面に追加」をタップ
  4. 名前を確認して「追加」
  5. ホーム画面にアイコンが追加されます。タップすると全画面のアプリとして起動します。
Chrome からでも追加は可能ですが、全画面のアプリ表示にはなりません。Safari での追加が推奨です。

6. 初期設定

  • 設定タブ → 手数料プリセットを確認（メルカリ 10%、ヤフオク 10% 等）
  • 必要に応じて販売先を追加
  • テスト登録 1〜2 件で動作確認

7. バックアップ（機種変更対応）

データは iPhone 内に保存されるため、機種変更時やデータ保全のため定期バックアップを推奨します。

  1. 設定タブ → バックアップ → 「JSONエクスポート」
  2. ダウンロードされた sedori-backup-YYYY-MM-DD.json を iCloud Drive や Gmail 下書きなどに保存
  3. 復元は同じ画面の「JSONインポート」から

8. 確定申告での使い方

  1. レポートタブで 対象年/月を選択
  2. CSV 出力（売上・経費・在庫）→ 確定申告ソフト（freee / マネーフォワード / やよい等）に取り込み
  3. PDF 出力（仕訳帳・総勘定元帳・損益計算書）→ 帳簿保管資料として保存（税法で 7 年保管）
免税事業者（事業登録なし副業）前提の複式簿記スタイルで出力されます。消費税の内税/外税分離はしません。

9. トラブルシューティング

Q: ホーム画面のアイコンをタップしても真っ白

A: Chrome で「ホーム画面に追加」した場合に起きることがあります。Safari から再度追加してください。

Q: 動作が遅い / データが肥大化した

A: 無圧縮画像を大量に登録すると IndexedDB が肥大します。設定 → バックアップで JSON エクスポート後、Safari のサイトデータを削除して初期化、インポートで復元できます。

Q: iPhone の画面下部（ホームバー）とボタンが重なる

A: Safari のタブ内で開いている場合に起きます。ホーム画面アイコンから起動してください。

Q: アプリを更新したのに古いバージョンが表示される

A: Service Worker のキャッシュが効いています。Safari → 設定 → サイトデータを削除、または「共有 → 再度ホーム画面に追加」で解決します。
